### ✅ 꼭 기억해야 할 내용

1. **위젯 선택 기준**
   - 상태 변경 없음 → StatelessWidget
   - 상태 변경 필요 → StatefulWidget
   - 데이터 공유 필요 → InheritedWidget/Provider

2. **라이프사이클 활용**
   - 초기화: `initState()`
   - Context 필요한 초기화: `didChangeDependencies()`
   - 상태 변경: `setState()`
   - 정리 작업: `dispose()`

3. **레이아웃 구성 팁**
   - Row/Column으로 기본 구조 잡기
   - Expanded로 유연한 공간 배분
   - Stack/Positioned로 겹침 효과
   - ListView/GridView로 스크롤 구현

4. **성능 최적화**
   - const 생성자 활용
   - 불필요한 setState 최소화
   - 적절한 위젯 타입 선택
